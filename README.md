# TaskManagerBot

Инструкции по запуску:

1. Установите .NET 6 SDK
2. Откройте папку в VS Code
3. Установите переменную окружения:
   - Windows (PowerShell):
     $env:TELEGRAM_BOT_TOKEN = "ТОКЕН"
   - Linux/macOS:
     export TELEGRAM_BOT_TOKEN="ТОКЕН"
4. В терминале:
   dotnet restore
   dotnet build
   dotnet run

По вопросам — обращайтесь.
